클래스 이름
	1. Java 클래스 이름은 대문자로 시작해야하고 여러 단어로 구성되는 경우 각 단어의 맨 앞을 대문자로 함
	2. 클래스의 이름과 파일명은 반드시 같아야함
	클래스명 변경하는 방법
	   1. 클래스 이름 변경 후, 파일명을 직접 바꾸기
	   2. 클래스 이름 선택-refactor-rename 선택 후에 클래스명 변경하기

변수 값 입력 - Scanner
	**Scanner 객체 사용시 라이브러리(import java.util.Scanner;)가 필요함
	만약 어떤 라이브러리가 필요한지 모를 때는 객체 선택-organize imports를 통해 자동으로 제공받을 수 있음

	next() : 문자열을 입력받는 메서드	== scanf("%s",...);
		**공백을 기준으로 첫 번째 단어만 입력받음
	A.equals(B) : == 의 역할로 문자열을 비교할 때 사용
