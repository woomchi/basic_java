클래스
: 서로 관련있는 데이터들을 하나의 단위로 묶은 것으로 사용자 정의 타입임(C언어 구조체와 동일)
	멤버 데이터: 클래스 내에 존재하는 변수, 메서드 등
	**각 파일은 하나의 클래스만 존재해야함
	**클래스는 int, String 처럼 하나의 타입의 역할이 됨
	=> 클래스는 타입이므로 클래스 자체에 데이터를 저장하는 것이 아니라 해당 클래스 타입의 변수에 저장하는 것임
	
	클래스 선언 및 객체 생성: 클래스타입 변수 = new 클래스타입(...);

	순서
	1. 클래스 타입의 변수 생성: 아직 객체가 생성되지 않았으므로 변수는 null을 가짐
	2. 객체 생성: 해당 클래스 타입의 객체를 생성하여 변수에 그 주소를 저장함
	3. 데이터 저장 : 클래스 타입 객체 필드에 각각 데이터를 저장함
=>클래스 타입은 설계도이고 참조변수와 객체가 실체임

클래스 배열
	선언
	클래스타입 [] 변수 = new 클래스타입[배열길이];
		**이때 각 클래스 배열은 따로 객체 생성해줘야함

프리미티브 타입과 클래스의 차이점
	프리미티브 타입: 선언된 변수에 값이 저장됨
	클래스 타입: 선언된 변수가 아닌 클래스 타입 객체에 저장됨 => 선언된 변수에는 그 객체의 주소가 저장됨(참조)

=> 프리미티브 타입은 변수 자체에 값이 저장되는 보통 변수이고 그 외 변수는 객체의 주소를 저장하는 참조 변수이다**
**배열은 프리미티브 타입이 아님
	만약 배열의 각 원소가 프리미티브 타입이더라도 배열 자체가 프리미티브 타입은 아님


클래스 변수 복사
: 클래스 변수의 객체 주소를 복사하는 것으로 기존 클래스 변수와 복사한 클래스 변수 모두 기존 클래스 변수의 객체를 지정하고 있음
